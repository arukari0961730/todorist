# グループ課題管理アプリ

## 概要

グループで取り組む課題を管理するためのWebアプリです。

課題名、詳細、担当者、締切日を登録し、カレンダー上で課題の期間や締切を確認できます。  
また、課題の詳細表示、完了切替、編集、削除もできます。

## 作成目的

グループ課題では、

- 誰が担当しているか分かりにくい
- 締切を忘れやすい
- 完了状況を共有しにくい

という問題があるため、課題を一覧で管理できるアプリを作成しました。

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage

## 主な機能

- 課題追加
- 課題名、詳細、担当者、締切日の登録
- localStorageによるデータ保存
- 月表示カレンダー
- 今日の日付の強調表示
- 月移動機能
- 今日へ戻る機能
- 課題の詳細表示
- 完了・未完了の切り替え
- 課題編集
- モーダル画面での編集
- 課題削除
