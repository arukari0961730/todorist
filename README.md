## 概要

このアプリは、ゲーム開発サークルでの制作作業を管理するためのWebアプリです。

現在は、タスクの追加・編集・削除・完了管理・カレンダー表示を中心に実装しています。  
今後は、複数グループでの作業管理、メンバー管理、チャット、画像共有、Pull Request情報共有などを追加し、サークル内の制作状況をまとめて確認できるアプリに発展させる予定です。

## 作成目的

ゲーム開発では、プログラム、デザイン、サウンド、企画など複数の役割に分かれて作業します。

その中で、

- 誰がどの作業を担当しているか分かりにくい
- 作業の進捗状況が共有しにくい
- 締切や確認待ちの作業を見落としやすい
- GitHubのPull Requestや修正内容の共有がばらばらになりやすい
- 画像や資料を含めた相談場所が必要になる

という問題があると考えました。

そこで、ゲーム開発サークル内で使える作業管理・情報共有アプリを作成することを目標にしています。

## 最終的な目標

最終的には、ゲーム開発サークルのメンバーがログインし、グループごとにタスクやチャットを管理できるWebアプリにすることを目標としています。

単なる個人用の課題管理アプリではなく、複数人でのゲーム制作に使える作業管理ツールとして発展させていきたいです。
